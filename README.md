# Data Manager App - Unit 5 Project

Your Unit 5 JavaScript project! You're going to build a mini data management app using arrays, objects, and functions.

## Getting Started

1. **Clone this repo** to your computer
2. Open the folder in your editor
3. Open `index.html` in a browser (with F12 open to see the console!)

## Your Job

### 1. Pick a topic
You get to choose what your data manager tracks! Some ideas:
- 📚 **Library** (books with title, author, year)
- 🎓 **Students** (name, grade, GPA)
- 🐾 **Pets** (name, species, age)
- 🎮 **Games** (title, platform, rating)
- 🎵 **Songs** (title, artist, duration)
- 🎬 **Movies** (title, genre, year)
- Or anything else you want!

### 2. Open `js/script.js`
Follow the 6 numbered steps. Each step has hints showing you the pattern.

### 3. Don't touch `css/styles.css`
Already done for you.

### 4. Don't touch `index.html` much
Just a card with a message telling you to check the console. Most of your output will go to console.log in this project (the BONUS 4 challenge lets you put stuff on the page if you want).

## What You'll Use

Everything from Unit 5:
- **Arrays** to hold your list
- **Objects** for each item
- **Array of objects** as your main data structure
- **Loops** (`for...of` or `for`) to go through items
- **Dot notation** (`item.name`) to get properties
- **`.push()`** to add new items
- **`.find()`, `.filter()`, or `.sort()`** to search or organize

Plus everything from earlier units:
- **Functions** with parameters and return values (Unit 4)
- **Template literals** for formatting (Unit 2)
- **console.log** for output (Unit 1)

## Requirements

- ✅ Array of AT LEAST 4 objects
- ✅ Each object has AT LEAST 3 properties
- ✅ AT LEAST 3 custom functions (one must return data)
- ✅ 2 different types of loops
- ✅ Uses `.push()` to add a new item
- ✅ Uses ONE of: `.find()`, `.filter()`, or `.sort()`
- ✅ Plenty of `console.log` to see what's happening

## Stuck?

1. Open the browser console (F12) - your output lives there
2. Getting `undefined`? Check your function returns
3. Can't access a property? Make sure you're using dot notation (`item.name` not `item[name]`)
4. Loop not working? Remember: `for...of` for arrays, `for...in` for single objects
5. Go back to your codealong file from class for reference

## When You're Done

Open your index.html in a browser with F12 console open. You should see:
- Your starting list printed out nicely
- A new item added (and the updated list)
- The result of your find/filter/sort

You got this! 💪
